# Vanshika Saxena Portfolio - PRD

## Original Problem Statement
Build a stunning personal portfolio website for Vanshika Saxena with scrapbook/editorial theme ("Tech meets poetry meets scrapbook diary"). React + Tailwind + Framer Motion frontend, FastAPI + MongoDB backend.

## Architecture
- **Frontend**: React (CRA) + Tailwind CSS + Framer Motion + EmailJS
- **Backend**: FastAPI + MongoDB
- **Fonts**: Playfair Display (editorial), Caveat (handwriting), Outfit (body)
- **Colors**: Cream #f5f0e8, Pink #f4a6c6, Charcoal #222222

## User Persona
- Vanshika Saxena: B.Tech CSE student, aspiring full-stack developer, speaker, debater, writer
- Target audience: Recruiters, internship reviewers, hackathon organizers, tech community

## What's Been Implemented (May 2026)
- 12 complete sections: Hero, About, Achievements, Tech Stack, Projects, Hackathons, Debate/Literature, Community, Timeline, Resume, Personal Board, Contact
- Working backend API (POST /api/contact, GET /api/contacts)
- EmailJS integration (placeholder keys - user needs to configure)
- Framer Motion scroll animations throughout
- Responsive design (desktop, tablet, mobile)
- Real user photos integrated (4 photos)
- Scrapbook aesthetic: polaroids, sticky notes, torn paper edges, grain overlay
